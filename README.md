These are my solutions to LeetCode problems written in Elixir.

All solutions have a corresponding test in [`test/solutions/`](test/solutions)

## Quick Start

```
git clone git@github.com:terenceponce/leetcode.git
cd leetcode
asdf install
mix test
```
